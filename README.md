# Wireless-Capacity-Predictor
A machine learning-powered web application that predicts wireless channel capacity based on Signal-to-Noise Ratio (SNR) and Bandwidth inputs.

## Features
- Predicts wireless channel capacity in real time
- Uses trained Neural Network / ML regression model
- Interactive Streamlit web interface
- Simulates telecom-based Shannon Capacity relationships

- Python
- Streamlit
- Scikit-learn
- NumPy / Pandas
- Machine Learning

- Linear Regression
- Decision Tree
- SVM
- Random Forest
- Neural Network (Best Model)
